Bienvenue sur le Projet gitspero

Ce projet est un package R avancé permettant de générer des rapports d'analyse basés sur des données régionales. Il inclut des fonctionnalités de création de rapports automatisés, une vignette explicative et une documentation complète avec pkgdown.

Fonctionnalités

  • calcul_distribution_age : Calcule la distribution des âges dans un dataset.
  • compter_nombre_d_adjoints : Compte le nombre d'adjoints dans un dataset.
  • compter_nombre_d_elus : Retourne le nombre total d'élus.
  • trouver_l_elu_le_plus_age : Trouve l'élu(e) le plus âgé(e).
  • summary.commune : Génère un résumé des statistiques d'une commune.
  • summary.departement : Génère un résumé des statistiques d'un département.
  • plot.commune : Génère un graphique des catégories socio-professionnelles des élus.
  • plot.departement : Génère un graphique des 10 professions les plus fréquentes.

Développement

Le package a été développé en suivant les étapes suivantes :

  1. Initialisation du projet avec Git et GitHub.
  2. Ajout des fonctionnalités principales et documentation.
  3. Création d'une vignette explicative.
  4. Configuration de la documentation avec pkgdown.
  5. Tests et validation du package.
